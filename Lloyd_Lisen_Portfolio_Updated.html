PROJECT 05
AI Lead Classifier + CRM Sync Workflow
Ultimate Watersports (WA) — n8n · Groq LLM · Google Sheets CRM · Railway
Live on Railway
The Problem
  • Booking inquiries arriving with no automated triage — team manually reading every message
  • No instant way to know if a lead was hot, warm, or cold without human review
  • Leads going cold while waiting for a manual sales response — lost bookings
  • No CRM — all lead data scattered, no structured follow-up system

The Solution
Webhook → Groq LLM → JS Parser → Sheets CRM → Webhook Response

  • Industry — auto-detected from message content
  • Intent — booking, inquiry, or quote request
